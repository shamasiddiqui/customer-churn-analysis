# Customer Churn Analysis & Retention Strategy

## 📌 Overview
This project analyzes customer churn behavior in a telecom dataset to identify key factors influencing customer retention.

## 📊 Dataset
- 1000 simulated customer records
- Features include age, tenure, contract type, monthly charges, complaints, and satisfaction score

## 🔧 Tools & Technologies
- Excel (Dashboard & Analysis)
- Python (Pandas, NumPy)
- SQL (Query Practice)
- Power BI (Guide included)

## 📈 Key Insights
- Customers with month-to-month contracts show higher churn rates
- Low satisfaction scores strongly correlate with churn
- Customers with more complaints are more likely to leave
- Long-tenure customers are less likely to churn

## 📊 Dashboard
The project includes an Excel dashboard showing:
- Churn rate
- Customer segmentation
- Key performance indicators (KPIs)

## 🤖 Machine Learning
- Implemented a basic Logistic Regression model to identify churn-driving features

## 📁 Project Files
- `generate_project.py` → Python script for dataset generation and analysis
- `Customer_Churn_Project_Shama.xlsx` → Final dashboard and analysis output

## 🎯 Business Impact
This project helps businesses identify high-risk customers and take proactive steps to reduce churn.
